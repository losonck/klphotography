# Portfolio Image Rights

**Copyright:** © KL Photography. All rights reserved.

All images in this directory were supplied by the photographer (the site owner) from
their own portfolio of client wedding work. They are NOT placeholder / stock imagery.
Do not redistribute, reuse, or republish outside of this site without written permission.

These files are build inputs (`src/assets/portfolio/`) — Astro's sharp pipeline emits
AVIF / WebP / JPEG variants to `dist/_astro/` at build time. Sharp strips all EXIF
(including any GPS coordinates of venues) before emission — see CONTRIBUTING-GALLERY.md
for details.
